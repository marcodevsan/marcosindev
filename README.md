### 🛠️ Principais Especialidades

* **OmniStudio:** Criação de experiências digitais guiadas e dinâmicas utilizando **FlexCards**, **OmniScripts**, **DataRaptors** e **Integration Procedures** para otimizar processos e integrar dados de forma declarativa e performática.
* **Sales Cloud & Service Cloud:** Implementação, customização e suporte de ponta a ponta para otimização de funis de vendas, automação de atendimento ao cliente, gestão de casos (Case Management) e consoles de serviço.
* **Desenvolvimento Apex & Lightning Web Components (LWC):** Desenvolvimento de código limpo e orientado a objetos, seguindo padrões de projeto (*Trigger Handler*, *Service Layer*), criação de componentes modernos e reativos com LWC e integrações via serviços REST/SOAP.
